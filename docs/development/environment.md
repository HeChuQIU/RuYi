# 开发环境

## 环境要求

- **Node.js**：>= 18.x
- **.NET SDK**：>= 8.0
- **PostgreSQL**：>= 14.x
- **Git**：>= 2.x

## 开发工具推荐

- **IDE**：Visual Studio / Visual Studio Code
- **数据库管理**：pgAdmin / DBeaver
- **API测试**：Postman / REST Client

## 环境配置

### Node.js 安装

访问 [Node.js 官网](https://nodejs.org/) 下载并安装最新 LTS 版本。

### .NET SDK 安装

访问 [.NET 官网](https://dotnet.microsoft.com/) 下载并安装最新版本。

### PostgreSQL 安装

访问 [PostgreSQL 官网](https://www.postgresql.org/) 下载并安装。

### 验证安装

```bash
# 检查 Node.js 版本
node --version

# 检查 .NET SDK 版本
dotnet --version

# 检查 PostgreSQL 版本
psql --version
```
