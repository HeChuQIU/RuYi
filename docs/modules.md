# 功能模块

## 核心模块

### 用户管理模块

- 用户注册、登录、认证
- 用户信息维护
- 基于角色的权限控制（RBAC）
- 用户角色分配和管理

### 模型管理模块

- **模型创建**：定义数据模型，一个模型对应一个数据库表
- **字段管理**：添加、编辑、删除模型字段
- **表结构管理**：自动创建和更新数据库表结构
- **模型关系**：配置模型之间的关联关系（一对一、一对多、多对多）

### 元数据配置模块

- **字段类型配置**：定义字段的数据类型（文本、数字、日期、布尔等）
- **验证规则配置**：设置字段验证规则（必填、长度限制、格式验证等）
- **页面展示配置**：
  - 列表页配置（显示字段、排序、分页）
  - 详情页配置（字段布局、分组）
  - 表单配置（字段顺序、布局方式）
- **字段编辑规则**：
  - 可编辑性控制（只读、可编辑、条件编辑）
  - 默认值设置
  - 字段联动规则（字段之间的依赖关系）
- **字段级权限控制**：配置不同角色对字段的查看和编辑权限

### SQL代码编辑器模块

- **SQL编辑器**：提供代码编辑、语法高亮、自动补全
- **SQL执行**：执行自定义SQL查询和脚本
- **自定义查询**：支持复杂查询、报表、统计分析
- **数据处理**：支持数据转换、批量操作
- **自定义业务逻辑**：支持存储过程、触发器定义
- **SQL代码管理**：保存、版本管理、权限控制

### 数据导入导出模块

- **数据导入**：
  - 支持 Excel (xlsx)、CSV、JSON 格式
  - 数据验证和错误提示
  - 批量导入功能
- **数据导出**：
  - 支持 Excel (xlsx)、CSV、JSON 格式
  - 支持自定义导出字段
  - 支持数据筛选和排序

### RESTful API模块

- **自动生成RESTful API端点**：根据模型和元数据自动生成
- **自动生成Controllers**：自动生成CRUD操作的Controllers
- **API文档生成**：自动生成RESTful API文档
- **API测试工具**：提供API测试和调试功能

## AI功能模块

### 元数据配置生成

- **智能模型生成**：根据自然语言描述生成数据模型和字段配置
- **元数据配置建议**：AI辅助生成字段类型、验证规则、展示配置等
- **配置优化建议**：对现有元数据配置提供优化建议

### SQL代码生成

- **SQL代码生成**：根据需求描述生成SQL查询、数据处理脚本
- **SQL代码补全**：智能代码补全和建议
- **SQL代码审查**：AI代码质量检查、性能优化建议

> 💡 **提示**：如需添加或删除功能模块，请直接修改上述列表。
